# Frontend Mentor - Workit landing page

## Project overview

This is a final solution of the **Workit landing page** from [Frontend Mentor](https://www.frontendmentor.io/challenges/workit-landing-page-2fYnyle5lu).

The site is built with plain **HTML and CSS** (no JavaScript) and is **responsive** to work on mobile and desktop.

## Live demo

- Live site URL: https://pettik-workit.netlify.app/

## What I did

- Built a responsive layout based on the design (desktop + tablet + mobile)
- Used **semantic HTML** (header / main / footer / article / section)
- Implemented **hover + focus states** for all interactive links and buttons
- Used local fonts from assets (Manrope + Fraunces) via @font-face
- Used **CSS Grid** and **Flexbox** for section layouts
- Added background and decorative SVG patterns matching the design

## Technologies

- HTML5
- CSS (custom properties, media queries, grid + flexbox)
- Images: provided assets (WebP / SVG)

## Project structure

- `index.html` – main page
- `assets/css/style.css` – styles
- `assets/images/` – images and icons
- `assets/fonts/` – local fonts

## What I learned / future improvements

- Better layout planning using CSS Grid combined with Flexbox
- Improve **accessibility** (ARIA attributes, better alt text, contrast)
- Add a simple form or modal for Apply for access (currently just an anchor)
